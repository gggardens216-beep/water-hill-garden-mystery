# Water Hill Garden Mystery

## Overview
A next-generation location-based AR puzzle game experience. Solve 9 mysteries across Water Hill Garden to unlock the golden seed hidden 100 years ago.

## Getting Started

### Installation
```bash
npm install
```

### Development
```bash
npm run dev
```
Visit http://localhost:5173 to play.

### Build
```bash
npm run build
npm run preview
```

## Tech Stack
- React 18.2
- Vite 4.4.5
- Tailwind CSS 3.3.3
- Lucide React Icons

## Features
- 9 interactive puzzle stages
- Camera AR scanning integration
- Real-time progress tracking
- Japanese language support with custom fonts
- Responsive mobile design
- Atmospheric storytelling
